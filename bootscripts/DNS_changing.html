Anonymous DNS's

To change your DNS server for all your machines, you can simply set it in your modem or router.
If this isn't the case, consider upgrading to open-source firmware (OpenWrt, ...).
Alternatively, you can set it per machine by:
openening the file dns.sh in the /bootscripts folder on the optional USB stick
then, changing the IP number by one of the numbers listed below
