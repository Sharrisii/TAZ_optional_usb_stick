The images folder can be used to save images for the feh-screensaver.
Images can be in .jpeg or .png format, and are best in the same resolution as your screen resolution (1024 x 768, ...). 
Images can be automatically fetched from USB devices since the TAZ livecd/usb has the autoFS package installed 
(=automounting enabled).
